# Data Analytics Project

Data analytics project showcasing customer behavior analysis using python, SQL and Power BI.

## Overview

This project demonstrates an end-to-end data analytics workflow, from data preparation to business insights and visualization. The analysis includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard development, and presentation of findings.

The goal is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

The project uses a structured dataset containing relevant business information for analysis.

The dataset is used to:

* Explore trends and patterns
* Clean and prepare data for analysis
* Perform SQL queries
* Create visualizations and dashboards
* Generate actionable business insights

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Inspected the data structure and data types.
* Checked for missing values and duplicates.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and summary statistics.
* Identified trends, correlations, and outliers.
* Created visualizations to better understand the data.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into PostgreSQL.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, joins, sorting, and aggregate functions to generate insights.

### 5. Dashboard Development

* Connected Power BI to the processed data.
* Built an interactive dashboard with key performance indicators (KPIs), charts, and filters.
* Designed visualizations to communicate insights clearly.

### 6. Reporting & Presentation

* Documented the analysis and findings.
* Prepared a professional presentation using Gamma.
* Summarized key insights and recommendations.

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Run the notebook to perform data loading, EDA, and data cleaning.
5. Import the cleaned dataset into PostgreSQL.
6. Execute the SQL queries.
7. Open the Power BI file to explore the dashboard.
8. Review the report and presentation for the project summary.
